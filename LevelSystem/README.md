# Draw LevelSystem

关卡系统版本入口。每个关卡 MVP 使用独立文件夹保存。

## Level 001

`Level_001/index.html` 实现首个完整关卡循环：

- 开始游戏与战士角色选择。
- 40c 开局，商店将卡牌、抽牌策略与武器分栏随机刷新。
- 只使用知识库图鉴中的中立卡牌与战士卡牌。
- 抽牌策略按实体牌管理，库存前六张按顺序循环执行；初始均为“筹备”。
- 收录哑铃、双刀、盾牌、匕首、大剑、炽焰之锤与石中剑，并按玩家排列顺序封装。
- 每个商店商品只能购买一次，可花费 2c 刷新整间商店。
- 采用 V1 战斗规则：抽牌、出牌、弃牌、武器封装与循环牌链。
- 以五份 Build 作为战力校准种子，通过同一战斗引擎模拟筛选，逐轮生成第 1–20 轮参考 Build 池。
- 玩家随机匹配同轮参考 Build；每场结算后记录玩家当轮 Build，并加入该轮本局数据池。
- 战斗支持 0.1x、0.5x、1x、2x 与 4x 倍速。
- 胜利奖励 15c，失败奖励 12c。
- 根据离开商店时的金币计算利息，每 5c 获得 1c，最多 5c。
- 累计三次失败后结束，并显示本局胜场。

## 运行

直接打开 `index.html` 进入版本库，再选择 Level 001。

Level 001 本身也是单文件页面，可以直接打开：

```text
Level_001/index.html
```
