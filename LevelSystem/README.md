# Draw LevelSystem

关卡系统版本入口。每个关卡 MVP 使用独立文件夹保存。

## Level 001

`Level_001/index.html` 实现首个完整关卡循环：

- 开始游戏与战士角色选择。
- 40c 开局，商店将卡牌、抽牌策略与武器直接分栏展示。
- 只使用知识库图鉴中的中立卡牌与战士卡牌。
- 战士拥有六个可替换的抽牌策略槽，按槽位顺序循环执行；初始均为“筹备”。
- 采用 V1 战斗规则：抽牌、出牌、弃牌、单手剑封装与循环牌链。
- 预置五轮模拟胜利 Build 作为基础数据库，每轮与对应轮次的玩家 Build 对战。
- 每场结算后记录玩家当轮 Build，逐步扩充本局数据库。
- 战斗支持 0.1x、0.5x、1x、2x 与 4x 倍速。
- 胜利奖励 15c，失败奖励 12c。
- 根据离开商店时的金币计算利息，每 5c 获得 1c，最多 5c。
- 累计三次失败后结束，并显示本局胜场。

## 运行

直接打开 `index.html` 进入版本库，再选择 Level 001。

Level 001 本身也是单文件页面，可以直接打开：

```text
Level_001/index.html
```
